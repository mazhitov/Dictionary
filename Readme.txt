Нужно скачать и установить Python3 для запуска проекта. 
https://www.python.org/ftp/python/3.9.2/python-3.9.2-amd64.exe

Если уже он уже установлен, то следовать следующим инструкциям. 

Инструкция для запуска текущего проекта: 

1. Нужно открыть командную строку или среду обработки по Python.
2. Перейти по директории D:\PYTHON\python programs\Dictionary\venv\Scripts\
3. Выбрать activate и нажать на Enter
4. Затем нужно перейти в папку \Dictionary\project\
5. Написать python manage.py runserver или (python3 manage.py runserver)
6. После скопировать ссылку с командной строки и вставить браузер.

Чтобы авторизоваться в админ панели, нужно прописать в конец ссылки '\admin'
